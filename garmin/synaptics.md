## Garmin - Synaptics Chipset

### Background

Garmin switched to a Synaptics chipset for the fenix 8, but a thorough analysis is yet to be completed.

There are lots of test activities that provide valuable insights into the performance of specific GNSS chipsets, including static testing, and various forms of kinematic testing. There are several categories of kinematic testing which include walking, driving, windsurfing, and wingfoiling.

Several other projects are currently taking precedence, so the Garmin evaluations are currently a background activity.



### Forerunner 970

Comparison of the Forerunner 255 and Forerunner 970 has proven to be inconclusive. There is too much variability in data from watches such as the Forerunner 255, so they are not a great as benchmark devices. Watches such as the Forerunner 255 may be deemed "good enough" for the green icon on GPS-Speedsurfing, but still be inadequate as a benchmark.

Since Jul 2026, I have received some data from the Forerunner 970 + Motion LCD. At this time, I am currently collecting this data so that analysis can be performed on larger volumes. This data set should prove to be very useful and allow the Forerunner 970 to be properly evaluated.



### fenix 8 Pro

I bought a fenix 8 Pro in Jul 2026 and have been collecting data on the water. I am currently collating this data so that analysis can be performed on larger volumes. This data set should prove to be very useful and allow the fenix 8 Pro to be properly evaluated.
